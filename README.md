# Coding Test Practice

コーディング試験対策用の学習アプリです。

## 目的

業務経験はあるものの、アルゴリズム問題や標準入力形式のコーディング試験に不慣れなエンジニアが、段階的に練習できることを目的としています。

## 技術構成

- Next.js
- TypeScript
- Tailwind CSS
- PostgreSQL
- Prisma
- Auth.js
- Monaco Editor
- Vitest
- Docker
- Github Actions

## 開発フェーズ

現在は Phase1 として、基本的な問題演習・コード実行・提出履歴の保存を実装します。
